## Cálculo do Factorial de um determinado "Inteiro" . 

**Este exercício exemplifica a aplicação dos seguintes itens:**
* Funções;
* Operador ternário "?".

## Notas
* O cálculo do factorial de um número inteiro é dado pela seguinte fórmula: 
                                         **n! = n x (n-1) x (n-2) x ... x n**
